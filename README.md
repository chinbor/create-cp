# create-cp

## usage

option `--template` or `-t` to declare which template to use

project name can be `.` ：the current directory

```bash
# npm 7+, 需要额外的双横线：
npm init cp my-app -- --t vitesse-lite

# yarn
yarn create cp my-app --t vitesse-lite

# pnpm
pnpm create cp my-app --t vitesse-lite
```
